DVOA

        Defense Adjusted Value over Average (DVOA), a measure created by Football Outsiders’ Aaron Schatz, can evaluate teams, units, or players. In our case, the DVOA measures each team's efficiency by comparing success on every play to a league average based on situation and opponent. Originally, on first down, a “successful play” was defined as attaining 45% of the total yardage needed. On second down, a “successful play” was defined as attaining 65% of total yardage needed. Finally, on third and fourth down, a “successful play” was defined as gaining a new first down. Every play either received a point value of 1 if a success or a 0 if not. Over time, the measure has been built upon and now a “successful play” considers specific scenarios, such as fumbles on play, big plays, red zone plays, or getting a touchdown on the play. Each of these scenarios are given either extra points, loss of points, or fractional points. Every single play in football gets a success value and that number is compared to the average success value of similar situations. These specific situations include distance, field location, time in game, etc.

Lastly, the DVOA is normalized to make it comparable across all teams and all years with an average offense and defense DVOA of 0%. The higher the DVOA offense percentage is, the more successful that team is. However, the DVOA defense is in terms of the average success of the offenses they are facing. Thus, a negative defense DVOA is equivalent to being a better defense.

Elo Score

        Elo score, as created by Arpad Elo, was originally designed to rank the skills of a chess player, but has since been applied to many other games and sports. The basic concept behind it is to continuously update a player or team’s Elo score after every game played. To update a team’s Elo score, we simply solve the following equations:

R(1) = 10 ^ (r(1)/400)

R(2) = 10 ^ (r(2)/400)

E(1) = R(1) / (R(1) + R(2))

E(2) = R(1) / (R(1) + R(2))

S(1) = 1 if team 1 wins, 0 if team 2 wins

S(2) = 0 if team 1 wins, 1 if team 2 wins

r’(1) = r(1) + K * (S(1) - E(1))

r’(2) = r(2) + K * (S(2) - E(2))

        where r(1) and r(2) are the original Elo scores for teams 1 and 2 respectively, and r’(1) and r’(2) are the updated scores. K represents the weighted value of each game. The value that is picked for K is somewhat subjective, but the goal is to find a K that approximates the significances of a game; if K is too small, Elo scores will change very slowly and thus make teams difficult to differentiate, if K is too big, Elo scores can jump all over the place and thus be too variable for accurate predictions. In our analysis, we first started with the 2019 season results and initialized each team to have the average Elo rating of 1500, and set the weighting variable K to be 15. Then, we performed the same calculations for the 2020 season, this time with the Elo scores found in the previous iteration and with a higher K value of 20. After this we update K once more to 25 and update the final team Elo scores for 2021. The logic behind the variable K is that the newest data is more important for predicting the results for games in the current season, but we assume that there is still information to be gathered from previous seasons since NFL teams should in general maintain the same relative amount of resources across seasons.

Methodology for Spread

        Spread is calculated by subtracting away points from home points. In this section, we will describe the different predictive models we created to predict point spread and explain why we chose the model we chose. For the purposes of this project, we aimed to minimize root mean squared error (RMSE).

Pythagorean Wins

We created a linear regression model to predict spread using Pythagorean wins. We trained this model on the first 8 weeks of 2020 and then used the later 8 weeks for testing to mimic the way our current information for the 2021 season is present. We found that home Pythagorean wins and away Pythagorean wins did not significantly increase the accuracy of our predictions, so we did not include Pythagorean wins in our spread predictions. It does make sense that Pythagorean wins does not predict score spread as well as it predicts wins because its purpose is to calculate expected wins.

DVOA

        By using the DVOA data on each team we created a linear regression model to predict spread. Due to the fact that the DVOA metric is updated periodically due to changing factors as seasons progress, we decided to only use the 2020 NFL season data in order to make sure we have the most effective version of the DVOA metric. Additionally, since we only have the first 8 weeks of data on 2021, we decide to divide the 2020 season data into two separate datasets one with the first eight weeks and another with the remaining weeks. To construct these models we first started by using cross-validation techniques and we divided our dataset into a training and testing dataset where 80% of our data was in the training dataset and the remaining 20% was in the testing dataset. By using the forward stepwise regression and stepwise regression methods on our training dataset we were able to build a model with a p-value of .01343. From here we then used the model built based on the training dataset to predict on the testing dataset that includes the games after week eight we got a p-value of .06951 and a RMSE of 9.6728. Using the same model to predict the spread of the games available in 2021 thus far we got a RMSE of 9.0303.

Elo Score

        To predict spread with our generated Elo ratings for each team, we created a linear regression model of interaction with our Elo scores. We trained this model on the first 8 weeks of 2020 and tested it on the latter 8 weeks. What we found was that Elo scores were insignificant if regards to spread, with p-values above 0.06 for each term and a model R^2 value of a measly 0.04. These poor results make sense; Elo scores only rate a team’s efficacy to win a game, not how well they win, so it follows that you wouldn’t be able to find out the point spread of a given game.

Chosen Model

        For predicting spread, we decided to use our DVOA model, since it gave us the best overall model on our train and testing data. Below is the complete regression equation derived from our model:

Spread = 9.5013 - 0.5179*Home_Offense_DVOA_Rank +0.3076*Away_Offense_DVOA_Rank - 0.3371*Home_Defense_DVOA_Rank

Methodology for Total

Total is the sum of home points and away points. In this section, we will describe the different predictive models we created to predict total points and explain why we chose the model we chose. For the purposes of this project, we aimed to minimize mean absolute error (MAE).

Pythagorean Wins

We created a linear regression model and a poisson regression model to predict total using Pythagorean wins. Just as before we trained on the first 8 weeks of 2020 and tested on the latter 8 weeks. Neither of them significantly increased the accuracy of our predictive model, so we did not include Pythagorean wins in our total predictions. It does make sense that Pythagorean wins does not predict total scores as well as it predicts wins because its purpose is to calculate expected wins.

DVOA

        In order to create a model to predict total, we used a similar methodology compared to how we constructed a model to predict spread. We used the DVOA data on each team to see which variables are significant at predicting total when using a linear regression model. As we did previously with spread, we decided to only use the 2020 NFL season data in order to make sure we have the most effective version of the DVOA metric. Additionally, since we only have the first 8 weeks of data on 2021, we decide to divide the 2020 season data into two separate datasets one with the first eight weeks and another with the remaining weeks. Once again we used cross-validation techniques to build models by dividing our dataset into a training and testing dataset where 80% of our data was in the training dataset and the remaining 20% was in the testing dataset. By using the forward stepwise regression and stepwise regression methods on our training dataset we got a model with a p-value of .006936. We then took our model to predict on the testing dataset and we got a p-value of .002121 and a MAE of 11.2141. We continued by using the model to predict on the 2021 data available and we got a p-value of .06366 and a MAE of 10.0783.

Elo Score

        To  predict total using Elo scores, we created two models, a linear regression model and a poisson regression model, since total is always positive. We trained and tested on the 2020 season just as before. Both of these models gave similar results to that of our Elo models for spread, except this time our p-values were far less significant, with values in the 0.4 range for the linear model and 0.2 range for the poisson model. These results also conform to our interpretation of what Elo scores are measuring.

Chosen Model

        The model we chose to predict total for the next 45 games of the 2021 season is a basic linear model that uses DVOA data variables. We chose this model as it provided us with the lowest MAE for the first 8 weeks of 2021. The model can be seen below:

Total = 59.5122 - .6742*Home_Offense_DVOA_Rank - .2326*Away_Offense_DVOA_Rank + .2360*Home_Defense_DVOA_Rank

Methodology for Result

The result of the game is classified as 1 if the home team wins and 0 if the home team loses. In this section, we will describe the different models we created to predict game result and explain the model that provided the highest classification accuracy.

Pythagorean Wins

        To predict the game results using our calculated Pythagorean wins for each team, we created two logistic regression models using home Pythagorean wins and away Pythagorean wins, one with interaction and one without. We trained both models for the first 8 weeks of the 2020 season. What we found was that the interaction model yielded insignificant results, but the regression without interaction provided p-values < 0.001, so we continued on using that model. With this model, we made predictions for the remaining 8 weeks of the 2020 season and found it had an accuracy of ~71.5%. Thus, Pythagorean wins are a good predictor for game results.  

DVOA

        As we did previously with creating models to predict total and spread, we used a similar methodology to create models to predict results. However, since results is a binary variable, we used a logistic regression model to predict results. Since the DVOA metrics are changed season to season, we decided to once again use the DVOA data for 2020 as we expect the latest version of the DVOA metric to be the best at predicting the results of a game in the following 2021 season. The same variables that were deemed significant predictor variables for predicting total and spread were used in the logistic regression model to predict results. Using the logistic regression models created we used them to predict on the 2021 data that is available and the model used for total was accurate 56% of the data observations and our model used to predict spread was right for 60% of the observations in our data.

Elo Score

        To predict the game results using Elo score, we first made a common transformation of both teams’ Elo score as follows:

P = 1 / (10^(-(elo home - elo away)/400) + 1)

This transformation of our Elo scores gives us what is typically thought of as the win probability for the home team. We then created a logistic regression model to predict the game results using this Elo win probability. We found that this model was statistically significant with a p-value < 0.001, so we used the calculated coefficient and intercept for our model to predict game results for the next 8 weeks of games for the 2020 season using the following equation:

Win Probability = e^(13.524 * P - 6.759) / ( e^(13.524 * P - 6.759) + 1)

        Doing this, we received a prediction accuracy of ~70.8%. Thus, we can see that Elo score is a good predictor for game results.

Chosen Model

        The model we decided to go with to predict results for the next 45 NFL games is a logistic regression model that combines both Elo scores and Pythagorean wins. We found that combining both scores together yielded an accuracy of ~72.7%, a modest improvement of both models individually. We found this result by again training our model on the first 8 weeks of the 2020 season and testing on the remaining 8 weeks. The complete regression equation is the following:

Win Probability = e^(8.2459* P + 0.2876*home pythagorean wins - 0.1911*away pythagorean wins - 4.8327) / ( e^(8.2459 * P + 0.2876*home pythagorean wins - 0.1911*away pythagorean wins- 4.8327) + 1)
